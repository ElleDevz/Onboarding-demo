# Onboarding UI Demo

A modern, mobile-first onboarding interface built with HTML, CSS, and JavaScript that matches the design from the provided image.

## Features

- **Three onboarding screens** with smooth transitions
- **Dark theme** with gradient backgrounds and glowing effects
- **Mobile-responsive design** that works on all screen sizes
- **Interactive navigation** with dots, buttons, and keyboard support
- **Touch/swipe gestures** for mobile devices
- **Smooth animations** and hover effects
- **Accessibility features** including keyboard navigation

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling including gradients, animations, and responsive design
- `script.js` - JavaScript for navigation, interactions, and gestures

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the screens using:
   - Click the "GET STARTED", "NEXT", or "CLOSE" buttons
   - Click the navigation dots at the bottom
   - Use arrow keys (left/right or up/down)
   - Swipe left/right on mobile devices
   - Press Escape to close

## Customization

- **Colors**: Modify the gradient values in `styles.css`
- **Content**: Update the text in `index.html`
- **Animations**: Adjust timing and effects in `styles.css` and `script.js`
- **Auto-advance**: Uncomment the last line in `script.js` to enable automatic progression

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
